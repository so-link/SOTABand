# 平方计算器

## 1. 功能概述

接收一个数字作为输入，计算并返回该数字的平方值（即 number²）。

## 2. 输入规范

| 参数名 | 类型 | 必填 | 默认值 | 说明 |
|--------|------|------|--------|------|
| number | float | 是 | 无 | 要计算平方的数字 |

## 3. 输出规范

| 字段 | 类型 | 说明 |
|------|------|------|
| status | string | 执行状态 (success/failed) |
| message | string | 结果说明 |
| data | dict | 输出数据，包含字段 result (float) 表示平方计算结果 |

## 4. 依赖环境

| 依赖 | 版本 | 用途 |
|------|------|------|
| Python | >=3.6 | 运行环境 |

## 5. 运行机制

### 5.1 执行流程

1. 接收输入参数 `number`
2. 校验参数是否为数值类型（若否则返回错误）
3. 计算平方（`number * number`）
4. 封装结果并返回

### 5.2 性能指标

- 预期执行时间: < 1ms
- 内存占用: < 1MB

### 5.3 错误处理

- 参数无效（非数字类型） → 返回验证错误：`status: failed, message: "输入必须为数字"`
- 执行异常（如超大数溢出） → 捕获并返回详细错误

## 6. 测试用例

### 6.1 测试数据描述

```json
{
  "input": {
    "number": 5
  },
  "expected_output": {
    "status": "success",
    "message": "计算成功",
    "data": {
      "result": 25
    }
  }
}
```

### 6.2 边界条件

- 输入0 → 应返回0
- 输入负数（如-3）→ 应返回9
- 输入浮点数（如2.5）→ 应返回6.25
- 输入极大值（如1e308）→ 视具体环境处理，可能返回无穷大或溢出错误
- 输入非数字字符串 → 返回验证错误

## 7. 调用示例

```python
result = execute(number=5)
# 返回: {"status": "success", "message": "计算成功", "data": {"result": 25}}
```
