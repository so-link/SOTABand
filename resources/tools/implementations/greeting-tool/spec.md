# 问候工具

## 1. 功能概述

返回字符串 "hello world"，用于问候演示或基础测试。

## 2. 输入规范

本工具无需输入参数。

| 参数名 | 类型 | 必填 | 默认值 | 说明 |
|--------|------|------|--------|------|
| - | - | - | - | - |

## 3. 输出规范

### 3.1 标准输出字段
| 字段 | 类型 | 说明 |
|------|------|------|
| status | string | 执行状态 (success/failed) |
| message | string | 结果说明 |
| output_format | string | **必须指定** — text |
| data | dict | 输出数据，格式由 output_format 决定 |

### 3.2 output_format 说明
- `text`: data 含 `{"text": "hello world"}` — 纯文本展示

## 4. 依赖环境

本工具无外部依赖。

| 依赖 | 版本 | 用途 |
|------|------|------|

## 5. 运行机制

### 5.1 执行流程

1. 工具函数被调用。
2. 直接返回预定义的字符串 "hello world"。
3. 封装为统一输出格式。

### 5.2 性能指标

- 预期执行时间: < 0.01s
- 内存占用: < 10MB

### 5.3 错误处理

- 本工具逻辑极简，不会产生错误；若因系统问题执行异常，将返回失败状态。
